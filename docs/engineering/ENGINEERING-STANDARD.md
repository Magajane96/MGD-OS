# MGD Engineering Standard 1.0

1. Arquitetura antes de implementação.
2. TypeScript em 100% do código.
3. Nenhum package deve depender de apps.
4. Comunicação entre domínios sempre por eventos.
5. Widgets leem Company State; não acessam banco diretamente.
6. Toda decisão arquitetural relevante deve gerar ADR.
